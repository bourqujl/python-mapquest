Python-MapQuest 0.1 Documentation¶

Python-MapQuest is a python module for interacting with the MapQuest API.

Versions¶

  • Version 0.1: Initial release, includes support for MapQuest geocoding service.

Tutorial¶

Instances of the MapQuest class serve as the main interface to the MapQuest API. Simply initiate a new instance of the class with your MapQuest API key to begin.

Example:

from mapquest import MapQuest

mq = MapQuest('mapquest_api_key')

location = '1555 Blake St,Denver,CO,80202'

results = mq.geocode(location)

# process results here

Classes¶

class mapquest.MapQuest(key, data=u'open', ssl=False, timeout=60, headers={})¶

Toplevel class for interacting with the MapQuest developer API.

Parameters:
  • key (str) – MapQuest developer API key.
  • data (str) – Specifies which MapQuest data to use. Options are either ‘open’ or ‘licensed’.
  • ssl (bool) – Specifies whether to use Secure Socket Layer (SSL) when fetching data from the MapQuest API (i.e. http vs https).
  • timeout (int) – Specifies how long the http request should wait (in seconds) for a response before timing out.
  • headers (dict) – Allows for the modification of the http request headers (i.e. specify a custom User-Agent).

Usage:

from mapquest import MapQuest
mq = MapQuest('mapquest_api_key')

Methods:

geocode(address, limit=-1, thumbnails=True, bounding_box=[])¶

Returns geocode data for a specified address. Useful for finding the relative latitude and longitude of an address.

Parameters:
  • address (str or dict) – The address the geocoder should look up.
  • limit (int) – Limits the number of results returned by the geocoder service. Specify -1 for unlimited results.
  • thumbnails (bool) – Specifies whether the geocoder service should return a URL to a map thumbnail image in the results for the location being geocoded.
  • bounding_box (list or tuple) – Moves any locations within the bounding box to the top of the results list when ambiguous results are returned (i.e. the provided address coresponds to multiple locations). See Bounding Boxes for more info.

The address may be specified as either a string or dictionary as follows:

address = '1555 Blake St,Denver,CO,80202'

address = {
    'street': '1555 Blake St',
    'city': 'Denver',
    'state': 'CO',
    'postalCode': '80202'
}
Returns:MapQuest Geocode Response
reverse_geocode(latlng, limit=-1, thumbnails=True, bounding_box=[])¶

Returns geocode data for a specified latitude and longitude. Useful for finding the relative address of a latitude and longitude.

Parameters:
  • latlng (str, list, or tuple) – The latitude and longitude the geocoder should look up.
  • limit (int) – Limits the number of results returned by the geocoder service. Specify -1 for unlimited results.
  • thumbnails (bool) – Specifies whether the geocoder service should return a URL to a map thumbnail image in the results for the location being geocoded.
  • bounding_box (list or tuple) – Moves any locations within the bounding box to the top of the results list when ambiguous results are returned (i.e. the provided address coresponds to multiple locations). See Bounding Boxes for more info.

The latlng may be specified as either a string or list as follows:

latlng = '39.7505568,-104.9996268'

latlng = [39.7505568,-104.9996268]
Returns:MapQuest Geocode Response
batch_geocode(locations, limit=-1, thumbnails=True, bounding_box=[])¶

Returns the geocode data for multiple addresses. Useful for finding the relative latitude and longitude of multiple addresses.

Parameters:
  • locations (list or tuple) – A list of addresses the geocoder should look up.
  • limit (int) – Limits the number of results returned by the geocoder service per address. Specify -1 for unlimited results.
  • thumbnails (bool) – Specifies whether the geocoder service should return a URL to a map thumbnail image in the results for the location being geocoded.
  • bounding_box (list or tuple) – Moves any locations within the bounding box to the top of the results list. See Bounding Boxes for more info.

Locations may be specified as either a list of strings or a list of dictionaries as follows:

locations = [
    '1555 Blake St,Denver,CO,80202',
    '2590 Pearl St,Boulder,CO,80302'
]

locations = [
    {
        'street': '1555 Blake St',
        'city': 'Denver',
        'state': 'CO',
        'postalCode': '80202'
    },
    {
        'street': '2590 Pearl St',
        'city': 'Boulder',
        'state': 'CO',
        'postalCode': '80302'
    }
]
Returns:MapQuest Geocode Response

Responses¶

Python-MapQuest returns MapQuest API responses as python dictionaries.

Sample Geocoding Input:

from mapquest import MapQuest

latlng = [39.7505568, -104.9996268]

mq = MapQuest('mapquest_developer_api')
mq.reverse_geocode(latlng)
